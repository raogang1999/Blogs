判断力，即判断的能力。判断是对于事物在某一方面下的定论。例如：命题：如果P，那么Q，对于命题的真假的确定即是一种判断，判断是主观的行为，客观来讲命题有真假，主观而言判断结果由判断者的认识决定，如果主客相符则是正确的判断，否则为不正确的判断。对于形式逻辑，一个命题必定有真或假的结果，但是对于辩证逻辑则不一定，辩证逻辑的例子：祸兮福所倚,福兮祸所伏，即一个事物可能包含正反两面，并且对于正反两面的判断都有道理。那么我们应该怎么运用我们的判断力呢？


农夫与蛇，讲述了一个农夫救活了一条冻僵的蛇，但反被蛇咬了而后丢掉性命。这个故事告诉了我们什么道理？可以从多个角度去探讨。农夫为什么要救蛇？因为农夫具有同情心，同情心是一种共情能力，即将自己代入对方情景的想象带来的感觉，将自己投射到对方处境中去，于是为了避免自己处于该状态（自己可能遇到类似的情况），本能地（生命自身想要延续的本能）想要给予帮助，于是乎农夫救了蛇，农夫完成了自己的职责。从蛇的角度，蛇为什么要咬农夫？因为蛇是冷血动物，也不具有理智，蛇苏醒的瞬间以为自己遇到了危险，于是本能的发起了攻击，导致了农夫的死亡。如果这样解读，双方都没有错误，但却导致了一方付出了生命的代价，对于人类而言，蛇会威胁到自己的生命，为了消除这个风险，这条蛇也应该被消灭，于是两方都丢掉了生命（报复，复仇）。也许从人与蛇的角度来说，显然人的生命是高于蛇的生命的，也就是期望人类自身能够得以存活，这证明了不同物种间生命的高低贵贱，各生命种类都有让自己得以延续的倾向，于是就得出了弱肉强食的自然规则。如果生命都平等，那么农夫救蛇，蛇咬农夫，蛇仍然拥有存活的权力，也就是人不去消除蛇的威胁，因为蛇只是按照本能行事，又由于生命平等，人也不应该继续迫害另一个生命。但是从生命延续的角度，蛇会继续导致人的生命的丧失，人就应该消灭蛇。可以得到，如果生命想要得以延续，就需要拥有剥夺一方生命的权力 ，如果有这样的能力那么显然生命间是不平等的。如果想要生命得以延续，人们也会让出部分权力，即在不平等的环境中生存。

以上是从两个不同的物种间讨论，如果蛇只是一个符号，一种象征意义，即坏的人，那么农夫象征好的人，好的必然会救其他人，坏的人为了获得利益必然会以各种手段去欺骗好的人。好的人为什么要救（帮助）坏的人？因为同情心。坏的人为什么要迫害好的人？因为利益。只要好的人有救坏的人的意愿，坏的人自然就想尽办法获利，通过伪装，欺骗等手段，从这个角度出发，好的人无法区分谁是真的需要帮助，谁是来迫害自己的，从博弈论的角度，好人如果帮助好人可能会有收益，如果帮助坏人会有坏处，如果都不帮助则没有损失也没有收益。从微观角度来说，作为个体，不帮助风险是最小的。从宏观角度而言，一个好人未被帮助会带来负收益，一个坏人未被帮助收益为0。于是整体而言，会越来越不好。

|     | 好人  | 坏人  |
| --- | --- | --- |
| 帮助  | +1  | -1  |
| 不帮助 | 0   | 0   |

因同情心而给予帮助是道德上的，为利益而欺骗有道德的人是物质上的，这两类人处于一个博弈的状态。从个体而言，不给予帮助当然是很好的选择，但如果从整个人类的幸福而言，不得不重新考虑如何做出选择。

这与判断力有关吗？如果能判断出对方是好与坏当然可以得到正向收益，如果判断不了，当然选择不予帮助是最优选择，为什么？

除了农夫与蛇的故事，还有一个小海龟的故事，游客看见破壳爬向海边的小海龟快被海鸥吃掉，于是游客将小海龟救了下来，抛向了大海，其他小海龟见安全下海，则纷纷从巢里爬出来，于是被海鸥叼走。本来是出于好心，却导致了更多的生命的丢失。

两个故事，本质上都是判断力的问题，从决策者来讲，选择帮助，会因为判断的正确性得到不同的收益，而如果不帮助，则收益为0，即没有损失。这似乎预示着不要提供帮助。

|      | 帮助  | 不帮助 |
| ---- | --- | --- |
| 正确判断 | +1  | 0   |
| 错误判断 | -1  | 0   |
除非拥有明断是非的能力，否则只能用“但行好事，莫问前程”来慰藉自己。

人想要得到自由，必然需要探索伦理。为了探索人性能达到的高度，必然是，路漫漫其修远兮。
